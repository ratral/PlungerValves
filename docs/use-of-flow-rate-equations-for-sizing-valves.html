7.1.1 Step 1. Specify the variables required to size the valve.

Service data : 1. Water pressure must be managed from 30 bars to 6 bars. 2. The control valve has a diameter of 250 mm and is smaller than the diameters of the pipe upstream (250 mm) and downstream (250 mm) of the valve.

physical constants:

  • Fluid : Water
  • Water Temperature : 23.3 °C
  • Altitude: 0 msal.
  • Atmosphere Pressure : 1.01325 bar
  • Vapor Pressure : 0.0286128 bar
  • Kinematic viscosity : 0.9430404 \((m^2/s)*1e-6\)

Service Conditions:

  • \(P_1 =\) 5.7 bar
  • \(P_2 =\) 4.8 bar
  • \(\Delta{P}\) 0.9 bar
  • \(q =\) 795.6 \(m^3/h\)
  • Diameter Pipe Upstream \(D_1\): 250 mm
  • Diameter Pipe Downstream \(D_2\): 250 mm

Valve Design (first approx.):

  • Valve: Plunger Valve VAG RIKO
  • Diameter Valve \(d\): 250 mm
  • PN-10
  • Control Element : Cylinder type Equal Percentage
  • Control Characteristic:

7.1.2 Step 2. Read the values of the selected Control Characteristic.

  • With the valve 100% open:
    • Zeta Value \(\zeta_{vs}\) : 3
    • Flow coefficient : \(K_{vs}\) : 1441.9 \(m^3 / h\)
    • liquid pressure recovery factor \({F_{L}}_{s}\) : 0.598

7.1.3 Step 3. Calculate the required flow coefficient (\(K_v\)).

Using the function (3.1):

\[ K_v = q \cdot \sqrt{\frac {(\rho / \rho_{0})}{\Delta P}} \] the Required \(K_v\) is 836.6 \(m^3/h\) and the relative flow coefficient is \(K_v/K_{vs}=\) 58%.

Entering the graph of \(k_V/K_{VS}\) with the value of 58% we find the position of the valve required for the flow of 795.6 \(m^3/h\) and \(\Delta{P}\) of 0.9 bar is 61.44%.

Figure 7.1: Inherent Valve Characteristics

Or it can be done directly with the \(K_V =\) 837 \(m^3/h\) value of on the \(K_V\) graph.

Figure 7.2: Flow coefficient

Knowing that the valve position is 61.44%, it is possible to read the Zeta Value \(\zeta_{v}\) and the Liquid Pressure Recovery Factor \({F_L}\).:

  • \(\zeta_{v}=\) 8.91
  • \({F_L}= 0.7\)
